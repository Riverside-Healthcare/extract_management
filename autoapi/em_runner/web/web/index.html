em_runner.web.web.send_ftp(task_id, run_id, file_id)[source]¶

Send file to FPT server specified in the task.

Url

/api/send_ftp/<task_id>/<run_id>/<file_id>

Parameters

  • task_id – id of task owning the connection

  • run_id – id of the run associated with the file

  • file_id – id of the file to send

Returns

json message or error

File is loaded from the backup SMB file server into a tempfile. The tempfile is deposited into the FTP location.
