em_web.web.project.project()¶

List all projects.

Url

/project

Returns

html page for projects. If no projects exist then redirect to new project page.

em_web.web.project.project_user(user_id)¶

List projects for a specific user.

Url

/project/user or /project/user/<user_id>

Parameters

user_id – id of project owner

Returns

html page for projects. If no projects exist then redirect to new project page.

em_web.web.project.project_get(project_id)¶

Project detail page.

Url

/project/<project_id>

Parameters

project_id – id of the project

Returns

html page for project details, or error page if project id was invalid.
