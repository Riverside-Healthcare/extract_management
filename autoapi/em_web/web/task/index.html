em_web.web.task.task_endretry(task_id)¶

Stop a task from performing the scheduled retry.

Parameters

task_id – id of task to stop

Returns

redirect to task details

First, remove the redis key for reruns Then, force the task to reschedule, which will clear any scheduled jobs that do not belong to the primary schedule.
